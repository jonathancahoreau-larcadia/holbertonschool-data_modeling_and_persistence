===============================================================================
                               INTRODUCTION AU SQL
===============================================================================

Ce projet introduit les bases de SQL (Structured Query Language) et du modèle
relationnel, fondations essentielles de la majorité des systèmes de gestion de
données modernes.

Une base de données relationnelle organise les informations sous forme de tables.
Chaque table est composée de :

  - Lignes (records) : entrées individuelles
  - Colonnes (fields) : attributs décrivant chaque entrée

Exemple : une table "books" peut contenir les colonnes :
title, author, price
Chaque ligne représente un livre unique.

SQL est le langage permettant de définir, interroger et manipuler ces données.


===============================================================================
                                   OBJECTIFS
===============================================================================

L'objectif de ce projet est de comprendre et manipuler les opérations CRUD :

  Create  -> INSERT   : ajouter de nouvelles données
  Read    -> SELECT   : récupérer des données existantes
  Update  -> UPDATE   : modifier des enregistrements
  Delete  -> DELETE   : supprimer des données

Le projet se concentre volontairement sur une seule table afin de permettre une
compréhension claire des mécanismes fondamentaux de SQL avant d'aborder les
relations entre tables.


===============================================================================
                                 CONCEPTS CLES
===============================================================================

1. Modèle relationnel
   - Organisation en tables
   - Structure lignes / colonnes
   - Types de données
   - (Selon contexte) clé primaire

2. Requêtes SQL fondamentales
   - INSERT INTO : ajouter une ligne
   - SELECT      : lire et filtrer les données
   - UPDATE      : modifier une ou plusieurs lignes
   - DELETE      : supprimer des données

3. Manipulation d'une table unique
   - Comprendre la logique de base
   - Écrire des requêtes simples et sûres
   - Préparer l'apprentissage des jointures
